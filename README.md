# Triple Espresso

# web_project_coffeeshop

En este proyecto desarrollé una landing page para una cafetería ficticia llamada Triple Espresso. La idea principal es tener una página sencillaen la que una persona pueda ver qué ofrece la cafetería o hacer una reservación.

La página está estructurada en varias secciones:

Encabezado (header) con el logo de la cafetería y un menú de navegación que lleva a las distintas partes de la página.

Una sección principal con el título y un texto introductorio que presenta la cafetería.

Una sección de recetas, donde se muestran textos e información acompañada por una lista de videos relacionados con bebidas o preparaciones de café.

Una sección de reservación, con un formulario donde el usuario puede ingresar sus datos para reservar una mesa.

Un footer con el logo, enlaces y espacio para información adicional, como redes sociales.

Este proyecto está hecho con HTML5 y CSS3.
Se utilizo:

HTML semántico para estructurar el contenido

Metodología BEM para nombrar las clases de CSS de forma ordenada.

Flexbox y posiciones relativas/absolutas para organizar el layout y acomodar elementos como el contenido principal, las tarjetas y el footer.

También se utilizaron fuentes externas y estilos personalizados para mantener el diseño lo más parecido posible a los brief proporcionado en el curso.

Planes de mejora del proyecto

Convertir la sección de recetas en un pequeño “catálogo interactivo”, donde el usuario pueda filtrar o cambiar entre diferentes videos o recetas con botones en un carrete.

Versión multidioma
Sería interesante agregar una opción para ver la página en español e inglés, usando una estructura que me permita cambiar el contenido de texto según el idioma seleccionado.
